## 🎮 交互设计

### 按钮文案

```

```

### 提示文案

```
- "Wallet not connected"
  → "WAGMI! Connect wallet first 🌈"

- "购买成功"
  → "Successfully minted! 🎉"

- "价格太高"
  → "Price too high for my taste 😤"

- "有人出价更高"
  → "Someone FOMO'd harder 💸"
```

```

- 升级: 进化动画
```
