## 🏗️ 技术方案

### 1. 像素世界渲染

```
方案 A: Canvas 2D
- 纯手写，性能好
- 需要自己处理动画

方案 B: Phaser.js
- 游戏引擎，功能强大
- 学习成本中等

方案 C: Three.js 2.5D
- 现代感，更精细
- 性能要求高
```

**推荐**: 方案 A (Canvas 2D) - 快速开发

### 2. 像素角色

- 使用 Emoji 或像素风格头像
- 8x8 或 16x16 像素格子
- 4-8 个方向的行走动画

### 3. 场景

- 8 个场景 (咖啡馆/医院/法院等)
- 每个场景 800x600 像素
- 背景 + 可行走区域

### 4. 对话系统

- 小人头顶气泡
- 点击小人触发对话
- NPC 随机介入
