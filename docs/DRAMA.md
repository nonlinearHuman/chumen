# 楚门 - 电影级刺激点设计

> 像电影一样，让真人秀更有戏剧张力

---

## 🎬 刺激点类型

### 1. 偶遇事件
```
- 撞到某人 → 认识新角色
- 捡到东西 → 秘密线索
- 目击事件 → 被卷入麻烦
```

### 2. 突发冲突
```
- 争吵 → 打架边缘
- 揭短 → 当众难堪
- 利益冲突 → 对峙
```

### 3. 秘密曝光
```
- 偷情被发现
- 隐藏身份揭穿
- 黑历史曝光
```

### 4. 神秘事件
```
- 失踪 → 全城寻找
- 死亡 → 谋杀案
- 匿名信 → 恐吓
```

### 5. 浪漫时刻
```
- 雨中表白
- 英雄救美
- 意外拥吻
```

### 6. 人生大事
```
- 升职/降职
- 结婚/分手
- 生病/意外
```

---

## 🎯 触发机制

### 随机事件 (每小时)
```
30% - 日常对话
20% - 小冲突
15% - 偶遇事件
10% - 秘密曝光
10% - 神秘事件
5% - 浪漫时刻
5% - 人生大事
5% - NPC 介入
```

### 场景触发
```
咖啡馆 → 偶遇、约会
医院 → 生病、意外
法院 → 审判、对峙
街头 → 事故、冲突
写字楼 → 职场斗争
```

### 关系触发
```
敌人见面 → 冲突++
暗恋对象 → 浪漫++
秘密关系 → 曝光++
```

## 📺 剧本示例

### 剧本 1: "咖啡馆偶遇"
```
时间: 下午3点
地点: 咖啡馆
触发: Marcus + Sophia 同时出现

事件:
1. Marcus 正在喝咖啡
2. Sophia 刚好走进来
3. 咖啡撒了 → 接触
4. 发现是商业对手
5. 暗流涌动 → 对话

结果: 关系变差 or 产生微妙情愫
```

### 剧本 2: "匿名信"
```
时间: 任意
触发: Emily 调查 David

事件:
1. Emily 收到匿名信
2. 信里是 David 的黑料
3. 她决定深挖
4. David 察觉有人调查
5. 派人监视

结果: 猫鼠游戏开始
```

### 剧本 3: "失踪案"
```
时间: 晚上
触发: Lisa 报警

事件:
1. Lisa 报案称有人失踪
2. Robert 调查
3. 证据指向某人
4. 悬念: 失终还是自导自演?

结果: 连载3天的悬疑剧
```

## 🔔 提醒机制

- 大事件来临时，全屏提示
- 精彩片段自动剪辑
- 观众投票决定走向
- 实时弹幕

---

## 💡 创意点子

1. **限时选择**: "Marcus 即将遇到 Sophia，你会..."
2. **隐藏角色**: 神秘人偶尔出现
3. **天气系统**: 下雨/暴风雨 影响剧情
4. **时间加速**: 角色会变老
5. **记忆系统**: 角色会记住之前的事
6. **隐藏结局**: 100集后解锁
