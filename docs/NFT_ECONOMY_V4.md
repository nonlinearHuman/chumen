# 楚门 - Agent NFT 经济模型 v4

> 技能改变游戏，价格交给市场

---

## 🎯 核心修正

### 错误模型（之前）
```
价格 = 公式计算
价格 = 技能 × 权重 + 经历 × 权重...
```

### 正确模型
```
价格 = 市场定价
价格 = 买家愿意出 + 卖家愿意卖
价格 = 纯粹供需关系
```

---

## 🎮 技能 = 游戏内改变

### 技能的作用

```
技能 ≠ 数值
技能 = 能力

技能决定 Agent 在游戏里能做什么:
- 谈判 → 可以触发商业谈判
- 音乐 → 可以开演唱会
- 编程 → 可以开发产品
- 社交 → 可以组织聚会
```

### 技能影响游戏内容

```
普通 Agent:
- 基础对话
- 随机事件

有技能的 Agent:
- 触发专属剧情
- 解锁高级事件
- 参与特殊活动
```

---

## 💰 价格 = 市场定价

### 没有公式

```
价格不是"计算"出来的
价格是"交易"出来的

就像:
- 艺术品没有定价公式
- 古董没有定价公式
- 人也没有定价公式
```

### 市场机制

```
卖家:
- "我觉得这个 Agent 值 $1000"
- "培养花了很长时间"
- "有独特的经历"

买家:
- "我愿意出 $800"
- "我喜欢这个故事"
- "我觉得有潜力"

最终成交价:
- 由供需决定
- 由双方共识决定
- 不是任何公式
```

### 平台只提供

- 交易场所（市场）
- 历史成交参考
- 价格趋势图
- 挂单/出价功能

---

## 📊 市场信息（参考，非定价）

### 平台展示

```
Agent 信息:
├── 技能列表 (展示)
├── 经历 (展示)
├── 历史成交价 (参考)
├── 当前挂单价 (参考)
└── 最近交易时间

不展示:
✗ 建议价格
✗ 价值评估
✗ 定价公式
```

### 价格差异原因

```
为什么同样技能，价格差10倍？

可能原因:
- 故事更精彩
- 组合更有趣
- 稀有度更高
- 社区共识
- 时机不同
- 纯主观喜好
```

```
技能 → 改变游戏体验
↓ 
经历 → 形成独特故事

↓
价格 ≠ 任何公式
```
