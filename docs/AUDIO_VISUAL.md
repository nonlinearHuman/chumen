# 楚门 - 视听设计方案

> 像素世界 + 电影级视听体验

---

## 🎨 视觉设计

### 1. 像素风格

```
角色规格:
├── 32x32 像素基础
├── 8方向行走动画 (每方向 4 帧)
├── 表情变化 (开心/生气/惊讶/思考)
└── 颜色区分阵营

场景规格:
├── 800x600 像素画布
├── 16色或32色调色板
├── 视差滚动背景
└── 动态装饰 (飘动的树叶/雨滴)
```

### 2. 场景视觉

| 场景 | 色调 | 氛围 | 特效 |
|------|------|------|------|
| 咖啡馆 | 暖黄 | 温馨 | 蒸汽、灯光闪烁 |
| 医院 | 冷白 | 紧张 | 蓝灯、心跳仪 |
| 法院 | 严肃灰 | 庄严 | 灯光、法锤 |
| 写字楼 | 冷蓝 | 高效 | 电梯、键盘声 |
| 街头 | 彩色 | 热闹 | 车流、人群 |
| 晚上 | 深蓝 | 神秘 | 路灯、影子 |

### 3. 天气系统

```
白天 ☀️ → 夜晚 🌙
└── 灯光渐变

晴天 → 雨天 🌧️
└── 雨滴特效 + 湿地面

晴天 → 暴风雨 ⛈️
└── 闪电 + 雷声 + 树枝摇晃
```

### 4. 角色外观

```
Marcus (CEO)
├── 穿西装 (黑/白)
├── 拿笔记本电脑
└── 表情: 自信/焦虑

Sophia (律师)
├── 穿职业装 (深蓝)
├── 拿文件夹
└── 表情: 冷峻/微笑

James (医生)
├── 穿白大褂
├── 拿听诊器
└── 表情: 温柔/严肃

Emily (记者)
├── 穿便装 (红)
├── 拿相机/笔记本
└── 表情: 好奇/紧张

Lisa (网红)
├── 穿时尚
├── 拿手机
└── 表情: 微笑/做作

David (投资人)
├── 穿西装(灰)
├── 拿雪茄/手机
└── 表情: 阴险/微笑
```

## 🔊 音效设计

### 1. 背景音乐 (BGM)

```
咖啡馆: ☕ 轻柔爵士 (loop)
├── 钢琴 + 萨克斯
├── 60 BPM
└── 音量 30%

医院: 🏥 环境音
├── 心跳仪 (滴-滴-)
├── 远处脚步声
└── 音量 20%

法院: ⚖️ 庄严音乐
├── 低沉弦乐
├── 法锤敲击
└── 音量 40%

街头: 🛣️ 城市噪音
├── 车流声
├── 喇叭声
└── 音量 30%

写字楼: 🏢 轻快节奏
├── 键盘声
├── 电梯声
└── 音量 25%

夜晚: 🌙 神秘音乐
├── 合成器氛围
├── 远处狗叫
└── 音量 20%
```

### 2. 事件音效

| 事件 | 音效 | 效果 |
|------|------|------|
| 对话弹出 | "叮" - 清脆提示音 | 音量 50% |
| NPC 介入 | "铛" - 金属敲击 | 音量 60% |
| 冲突开始 | "轰" - 低沉鼓声 | 音量 70% |
| 浪漫时刻 | "叮~" - 温柔风铃 | 音量 40% |
| 悬疑 | "滴答" - 时钟 | 音量 50% |
| 意外 | "啊!" - 惊讶声 | 音量 60% |
| 下雨 | "沙沙" - 雨声 | 音量 30% |
| 雷暴 | "轰!" - 雷声 | 音量 80% |

## 🎬 镜头系统

### 1. 镜头类型

```
远景 (Wide)
└── 展示整个场景

中景 (Medium)
└── 2-3人对话

特写 (Close-up)
└── 角色表情

过肩镜头 (Over-shoulder)
└── 对话常用

俯视 (Bird's eye)
└── 展示位置关系
```

### 2. 镜头切换

| 场景 | 镜头 |
|------|------|
| 日常对话 | 中景 → 特写 |
| 冲突 | 快速切换 |
| 浪漫 | 慢速推近 |
| 悬疑 | 俯视 + 特写 |
| 大事件 | 全景 → 特写 |

## ✨ 视觉特效

### 1. 对话气泡

```
普通对话:
├── 白色圆角气泡
├── 黑色文字
└── 淡入动画

重要对话:
├── 金色边框
├── 发光效果
└── 放大弹入

NPC 介入:
├── 闪烁边框
├── 震动效果
└── 音效: 铛!
```

### 2. 场景特效

```
咖啡馆:
├── 蒸汽从杯子升起
├── 灯光微微闪烁
├── 窗外阳光

医院:
├── 心跳仪闪烁
├── 护士走动
├── 蓝绿光

法院:
├── 光束从窗户照入
├── 灰尘漂浮
├── 法锤光晕

暴风雨:
├── 雨滴落下
├── 闪电照亮
├── 树枝摇晃
```

### 3. 转场效果

```
场景切换:
├── 淡入淡出 (1秒)
├── 滑动 (0.5秒)
└── 闪烁 (0.3秒)

时间切换:
├── 白天→夜晚: 渐变
├── 日出日落: 色彩变化
└── 快进: 画面闪烁
```
