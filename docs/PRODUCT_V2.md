# 楚门 - 产品定位更新

> 基于新的目标用户重新设计

---

## 🎯 核心定位

**不再是"养成游戏"，而是"Web3 精英俱乐部"**

类似 BAYC (无聊猿)、Azuki 的逻辑：
- 购买的不是"宠物"
- 而是"社交身份"
- 进入某个圈子的"入场券"

---

## 👥 目标用户分析

### 1. 名人与潮流人士
```
特征:
- 有钱有名，追求前沿
- Web3 新手，但想入圈
- 需要"社交入场券"

痛点:
- 不知道买什么NFT显示身份
- 怕被割韭菜

解决方案:
- 品牌故事包装
- 名人站台
- 精英俱乐部会员权
```

### 2. 投资者与投机者
```
特征:
- 懂 Web3，看重回报
- 会分析项目基本面
- 关注商业扩展

痛点:
- 担心跑路
- 担心没有实际价值

解决方案:
- 游戏、元宇宙土地规划
- 商业扩展路线图
- 代币经济学
```

### 3. 寻求社区归属
```
特征:
- 认同某种叙事/文化
- 想要圈子认同感
- 愿意参与社区

痛点:
- 单纯持有很无聊

解决方案:
- 线下俱乐部活动
- 线上 Discord 社群
- 专属特权
```

### 4. 潮流追随者
```
特征:
- 跟风，看热度
- 想要显得"在潮流尖端"
- FOMO 心理

痛点:
- 不知道买什么

解决方案:
- KOL 推广
- 饥饿营销
- 热度话题
```

## 💎 产品设计

### 核心: "身份 NFT"

```
不像"宠物养成"
而像"俱乐部会员"

持有 = 身份象征
持有 = 进入圈子的门票
持有 = 社区特权
```

### NFT 层级

| 层级 | 数量 | 价格 | 权益 |
|------|------|------|------|
| Founder | 10 | $99,999 | 俱乐部董事 |
| Platinum | 100 | $9,999 | 核心会员 |
| Gold | 1,000 | $999 | 正式会员 |
| Silver | 10,000 | $99 | 观察员 |

## 📈 商业模式

### 收入来源

| 来源 | 模式 | 预估占比 |
|------|------|----------|
| NFT Mint | 首次销售 | 60% |
| 交易版税 | 5-10% 二次销售 | 15% |
| 游戏内购 | 土地/道具 | 15% |
| 周边产品 | 限定商品 | 5% |
| 会员费 | 年费订阅 | 5% |

### 估值模型

```
假设:
- Founder 10个 × $100K = $1M
- Platinum 100个 × $10K = $1M
- Gold 1,000个 × $1K = $1M
- Silver 10,000个 × $100 = $1M

总计: $4M (首次销售)

+ 二次销售版税 (持续)
+ 游戏收入 (未来)
+ 周边 (未来)
```
