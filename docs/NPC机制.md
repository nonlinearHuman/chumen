# 楚门 - NPC 爆点机制

> NPC 的任务是让"真人秀"好看，像电影一样制造爆点

---

## 🔥 核心逻辑

```
Agent 自由聊天
     ↓
NPC 观察 + 等待时机
     ↓
发现"戏剧苗头" → 立刻介入
     ↓
制造冲突 / 偶遇 / 谣言 / 事件
     ↓
剧情爆点产生！
```

---

## ⚡ 触发条件

### 1. 情绪触发
```
当 Agent 对话中出现强烈情绪词：
- 争吵："生气"、"愤怒"、"滚"
- 暧昧："喜欢"、"心跳"、"脸红"
- 秘密："别告诉"、"不能说"、"被发现"
→ NPC 立即介入
```

### 2. 关系触发
```
当两个 Agent 关系发生变化：
- 敌人变朋友
- 朋友变敌人
- 单相思
→ NPC 推波助澜
```

### 3. 沉默触发
```
当某个 Agent/区域长时间无动静
→ NPC 主动搞事情
```

### 4. 随机触发
```
每 10-15 分钟
→ NPC 随机制造小惊喜
```

### 5. 大事件（每周一次）
```
周末
→ NPC 联合搞个大事件（悬疑/冲突/告白）
```

## 🔄 循环流程

```
1. Agent 自由互动（5-10分钟）
      ↓
2. Observer 检测"可介入点"
      ↓
3. Trigger 决定干预方式
      ↓
4. NPC 介入 → 爆点！
      ↓
5. 评估师判断效果
      ↓
6. 回到 1
```

## 📝 实际效果示例

**场景：咖啡馆，Marcus 独自工作**

```
[自由聊天]
Marcus: (内心) 最近融资不太顺，David 那边施压...
     [NPC 观察]
     Observer: 检测到负面情绪，可介入

     [NPC 决策]
     Trigger: 派老李去

[爆点！]
老李: "小伙子，我刚拉了个客人，听口气是你投资人？
       那家伙接电话说...'得逼他一把，不行就撤'"

Marcus: (警觉) 什么？！

老李: "具体我不懂，但你小心点"

→ 观众：悬念来了！
→ 下一集：Marcus 调查 David
```

```typescript
interface NPCTask {

  urgency: 'low' | 'medium' | 'high';
}
```
