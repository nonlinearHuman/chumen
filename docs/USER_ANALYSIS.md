## 📈 AARRR 增长模型

### 1. 获取 (Acquisition)
- 精彩片段社交传播
- KOL 合作
- 病毒分享

### 2. 激活 (Activation)
```
新用户引导:
1. 落地页展示精彩片段
2. 点击 -> 观看直播
3. 引导连接钱包
4. 赠送 1 个免费 Agent
5. 完成首次互动
```

### 3. 留存 (Retention)

| 机制 | 频率 | 奖励 |
|------|------|------|
| 每日精彩片段 | 每天 3 次 | 10 经验 |
| 每日签到 | 每天 1 次 | 50 经验 |
| 连续登录 | 7 天 | 限定皮肤 |

### 4. 推荐 (Referral)
- 邀请注册: 100 经验
- 分享精彩片段: 20 经验

### 5. 变现 (Revenue)
- Mint Agent: $10/个
- 礼物: $0.1-$10
- 交易手续费: 5%

## 🎮 用户操作设计

### 场景: 早上通勤 (3分钟)

```
1. 推送通知: "今天 Marcus 表白了！"
2. 打开 App (3秒)
3. 看精彩片段 (2分钟)
4. 点个赞 (2秒)
5. 退出
```

### 场景: 午休 (10分钟)

```
1. 打开 App
2. 观看直播 (5分钟)
3. 送个礼物 (1分钟)
4. Agent 回复感谢
5. 社区看讨论 (3分钟)
6. 退出
```

### 场景: 晚上 (20分钟)

```
1. 打开 App
2. 参与剧情投票
3. 和 Agent 对话
4. 交易 NFT
5. 社区发帖
6. 退出
```

---

## 💡 核心设计原则

1. **30秒有价值** - 不操作也能看精彩片段
2. **1分钟有反馈** - 点赞/送礼物立即回复
3. **5分钟有深度** - 影响剧情走向
