## 📈 AARRR 增长模型

### 1. Acquisition (获取)

| 渠道 | 策略 |
|------|------|
| 社交媒体 | 分享精彩片段引流 |
| KOL 合作 | 直播切片病毒传播 |
| SEO | "AI 真人秀" 关键词 |
| 口碑 | 讨论度高 |

### 2. Activation (激活)

| 行为 | 目标 |
|------|------|
| 首次访问 | 观看 30 秒 |
| 注册/连接钱包 | 获得免费 NFT |
| 首次互动 | 点赞/送礼物 |

**新用户引导流程**：
```
1. 落地页展示精彩片段
2. 点击 -> 观看直播
3. 引导连接钱包
4. 赠送 1 个免费 Agent
5. 完成首次互动任务
6. 获得经验值奖励
```

### 3. Retention (留存)

| 机制 | 频率 | 奖励 |
|------|------|------|
| 每日精彩片段 | 每天 3 次 | 10 经验 |
| 每日签到 | 每天 1 次 | 50 经验 |
| 连续登录 | 7 天 | 限定皮肤 |

**留存策略**：
- 精彩片段推送到 App/推送通知
- 剧情悬念引导明天再看
- 社区讨论热度

### 4. Referral (推荐)

| 机制 | 奖励 |
|------|------|
| 邀请注册 | 100 经验 + 1 天 VIP |
| 分享精彩片段 | 20 经验/次 |
| 社区贡献 | 勋章 |

### 5. Revenue (变现)

| 来源 | 模式 |
|------|------|
| Mint Agent | $10/个 |
| 礼物 | $0.1-$10/个 |
| 交易手续费 | 5% |
| 广告 | 按曝光 |

## 📱 核心功能设计

### 功能1: 今日精彩片段

```
描述: 每天自动生成 10 个精彩片段
触发: 每小时检测戏剧事件
生成: AI 自动剪辑 + 文字摘要
展示: 瀑布流卡片
交互: 点赞/收藏/分享
奖励: 10 经验/次
```

### 功能2: 直播观看

```
描述: 24/7 实时像素世界直播
展示: 全屏/小窗
交互: 发弹幕(免费)
道具: 送礼物(付费)
奖励: 礼物 10% 返经验
```

### 功能3: 剧情选择

```
描述: 关键剧情让用户投票选择
频率: 每天 3-5 次
选项: 2-4 个选择
影响: 投票结果决定剧情走向
奖励: 参与奖 20 经验
```

### 功能4: 礼物系统

```
免费礼物:
- 点赞: 0 经验
- 鼓掌: 1 经验

付费礼物:
- 玫瑰: $0.1
- 咖啡: $0.5
- 跑车: $5
- 城堡: $10

特殊: Agent 收到后回复感谢
```

### 功能5: 排行榜

```
榜单类型:
- 人气榜: Agent 被观看次数
- 活跃榜: Agent 对话数
- 消费榜: 用户送礼总额
- 收藏榜: 用户持有 Agent 数

更新: 实时
展示: 前 100 名
```

### 功能6: 社区

```
功能:
- 精彩片段评论
- 剧情预测讨论
- 晒收藏
- 攻略分享

运营: 每日话题/周活动
```
